# LearnSphere.ai

LearnSphere.ai is a revolutionary platform designed to help every learner master any skill through adaptive AI learning, personalized study plans, and intelligent tutoring.

## Features

- **AI Personalized Roadmaps**: Custom study paths adapted to your pace and goals.
- **Smart Revision**: Spaced repetition engine to eliminate forgetting.
- **Adaptive Practice**: Quizzes that adjust difficulty based on your performance.
- **Coding Playground**: Real-time AI feedback on your code syntax and logic.
- **Mock Interviews**: Practice technical interviews with our voice AI coach.
- **Career Guidance**: Data-driven insights to prepare you for top tech roles.

## Getting Started (Local Development)

To run this application locally on your machine, follow these steps:

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Start the development server**:
   ```bash
   npm run dev
   ```

3. **View the app**:
   Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Running in Antigravity

To run and iterate on this application within the Antigravity environment:

1. **Prompt the Agent**: Simply describe the features you want to add or modify in the chat.
2. **Automatic Build & Preview**: The Antigravity agent will automatically write the code, install any necessary dependencies, and compile the applet.
3. **Live Preview**: The built-in preview will automatically refresh to show your latest changes once the agent completes its turn.
4. **Iterate**: Continue prompting the agent to refine your application in real-time. No manual terminal commands required!
